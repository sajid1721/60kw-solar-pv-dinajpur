# 60 kW Solar PV Power Plant Design – Dinajpur, Bangladesh

## 📌 Project Overview
This repository presents the design and feasibility analysis of a **60 kW Solar Photovoltaic (PV) Power Plant** intended for a multi-unit residential building in Dinajpur, Bangladesh. The project evaluates the technical, environmental, and economic feasibility of implementing solar energy for sustainable electricity generation.

The study includes:
- Load demand analysis
- Resource assessment
- PV system sizing
- Component selection
- Performance optimization
- Economic analysis
- Environmental & social impact assessment

---

## 🌍 Project Location
📍 Dinajpur Sadar, Dinajpur, Bangladesh  
- Latitude: 25°45'19" N  
- Longitude: 88°41'32" E

The selected site has favorable solar irradiance conditions and sufficient rooftop/land area suitable for solar PV installation.

## ⚙️ System Specifications

| Component | Specification |
|---|---|

| Solar Technology | Monocrystalline Solar Panels |
| Number of Panels | 134 Panels |
| Inverter Type | Grid-Tied Inverter |
| Battery System | Tesla Powerwall 2 |
| Charge Controller | Schneider Electric MPPT |
| Estimated Project Life | 25 Years |
